GeneraPassword

GeneraPassword is an free application to create strong passwords starting from simple keys easy to remember.
Thanks to a script encryption Javascript and an HTML form, you can generate alphanumeric passwords (with or without special characters), alphabetic or numeric, the length we wish.

To create your secret code (PIN), just enter a key pair in the fields Secret Key and Password, choose the type of password you want to get (i.e. alphanumeric, alphabetic, numeric only, alphanumeric with special characters), specify its length and move to the last field Code.

The system will automatically generate a masked password, resulting from the previously entered parameters, which can be shown using the appropriate command [ Show ], or it can be copied "in the background" with normal functions of the device such as : Select, Copy and Paste, directly where needed.

Note: The code generated (PIN) is neither sent nor saved anywhere and will be deleted once you close the page or the application.

The advantage of this system is that we are able to adopt a rule set itself to generate a password, complex and not associated to any possible word, which if necessary can be obtained simply by utilizing the rule described above.

For example, we can set as Secret Key the name of the site or service on which we want to use the password (i.e. "Bank" or "Gmail" or "Twitter") and enter as Password a word easy to remember (or even the old and "usual" password). Then all you need is to set the length and type of code to be generated and the result will be obtained.
To regenerate the secret code, will be enough to remember the name of service and the old password.

Tip: To create a different secret code (PIN) every month, you can set in the rules to insert in the keys the number of the month so that automatically a different password can be generated.
